# Search_invocador

## Getting Started

### Install [Flutter SKD](https://flutter.dev)

Run `flutter pub get`

```bash
$ flutter pub get
```
Run `flutter run` or `flutter install` (with a device or emulator connected)

```bash
$ flutter run
```
```bash
$ flutter install
```
## Data

This app is using [API-Riot-Data](https://github.com/Lucasmaia435/API-Riot-Data).
